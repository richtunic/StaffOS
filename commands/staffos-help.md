# SEOS Help

Comandos disponibles:

- seos-review: revisar diff actual.
- seos-audit: auditar proyecto completo.
- seos-security: revisar seguridad.
- seos-handoff: actualizar documentación para el siguiente agente.
- seos-help: mostrar ayuda.

Modos recomendados:

- Lite: cambios pequeños.
- Full: tareas normales.
- Ultra: auditoría estricta, seguridad y arquitectura.
