# Antigravity Adapter

Este adaptador define cómo configurar las directrices de SEOS para Antigravity.

## Configuración

Antigravity puede cargar habilidades o skills especificadas en `skills/seos/SKILL.md`.

Para usar SEOS en Antigravity:

1. Coloca `skills/seos/SKILL.md` en tu espacio de trabajo.
2. Agrega instrucciones en `AGENTS.md` o en las reglas de tu conversación para que Antigravity lea `skills/seos/SKILL.md` como skill predeterminada.
