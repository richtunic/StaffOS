# OpenCode Adapter

Este adaptador define cómo OpenCode debe configurar las directrices de SEOS.

## Configuración

Para usar SEOS en OpenCode:

1. Agrega el contenido de `skills/seos/SKILL.md` en tu configuración de prompt de sistema en OpenCode.
2. Asegúrate de incluir el archivo `AGENTS.md` en la raíz del proyecto para que los agentes lo lean al iniciar.
