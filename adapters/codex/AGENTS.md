# Codex Adapter

Este adaptador sirve para cargar la skill SEOS en Codex.

## Configuración

Para usar SEOS en Codex:

1. Copia el archivo `skills/seos/SKILL.md` y `AGENTS.md` a la raíz del repositorio.
2. Agrega una directiva en tu prompt o reglas de Codex para hacer referencia a las directrices de `skills/seos/SKILL.md`.
