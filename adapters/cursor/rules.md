# Cursor Adapter

Este adaptador define cómo configurar las reglas de SEOS en Cursor.

## Configuración

Cursor utiliza un archivo `.cursorrules` en la raíz del proyecto para aplicar instrucciones personalizadas al chat y a Composer.

Para usar SEOS en Cursor:

1. Crea o modifica el archivo `.cursorrules` en la raíz del proyecto.
2. Copia y pega el contenido de `skills/seos/SKILL.md` dentro de `.cursorrules`.
