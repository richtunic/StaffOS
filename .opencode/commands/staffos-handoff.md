# SEOS Handoff

Actualiza el estado del proyecto para el siguiente agente.

Actualizar:

- docs/CHANGELOG_AI.md
- docs/DECISIONS.md
- docs/HANDOFF.md
- docs/AI_CONTEXT.md si cambió arquitectura o stack.

HANDOFF debe incluir:

```txt
Última tarea:
Archivos modificados:
Estado actual:
Qué funciona:
Qué falta:
Riesgos pendientes:
Siguiente paso recomendado:
Notas para el siguiente agente:
```
